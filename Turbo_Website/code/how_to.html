In order to get Turbo to run on your system you should need to know about the following things:

  • You know how to write C++ Code
  • You know what a compiler is
  • You know how to compile C++ Code

The following points are good to have but are not really necessary:
  • You got some knowledge about Python
  • You know how to compile programs with a makefile

The first step to include Turbo into your game you need to download one of the following resources.If you are familiar with the listed points above you should be able to setup Turbo on any system with any compiler, otherwise it is important that you use the MinGW-Compiler on Windows:

Only sources: Link *.cpp and *.h files
  • include
  • source
Sources and Libraries: Link *.cpp and *.h files, *.a and *.dll files
  • include
  • lib
  • source


If you have downloaded the package you wanted, it is time to extract the contents of the .zip file into your project. To check if everything is okay you can execute this command via the console:
