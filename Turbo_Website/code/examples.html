In order to use Turbo as your game engine you have to create a class and inherit the class 'Application' from the module 'turbo'. If you followed the install instructions the turbo module should be located at './include/turbo/'. The following functions are pure virtual member functions of the Application class and have to be implemented:
  • registerCallback()
  • run()
