Turbo ist eine Spiele Engine die vor allem für 2D Spiele gedacht ist. Durch die low level Anbindung an SDL2 ist Turbo vor allen Dingen Platform unabhängig. Turbo zielt vor allem auf Programmieranfänger und Programmierer ab die keine Zeit mit Kleinigkeiten verbringen wollen. Auch für Fortgeschrittene bietet Turbo eine Script Anbindung für Python(34). Grundsätzlich ist Turbo ein Wrapper für SDL, damit unwichtige Funktionen versteckt werden und eine Klassenabsierte Anwendung geboten wird.
Das Ziel von Turbo ist es einfachen, übersichtlichen und gut wartbaren Code anzubieten.

Features:

  • Input detection: Keyboard, Mouse, Controller, (Finger, up to come...)
  • Easy Entity/Component system via XML
  • openGL support
  • Python scripting support
  • open source
